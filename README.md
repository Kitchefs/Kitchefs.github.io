# Kitchefs

A new, beta version, Kitchefs app for easier management.

## Using

# Development

```
git clone https://github.com/Kitchefs/beta.git
cd beta
npm install
npm run dev
```

# Building

```
git clone https://github.com/Kitchefs/beta.git
cd beta
npm install
npm run export
```

# Adding a New Recipe

Edit the `./src/routes/recipes/_recipe.js` file and add the required information.
